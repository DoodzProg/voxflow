Features

Everything you need.
Nothing you don't.

A focused tool that does one thing exceptionally well.

Hotkey-driven dictation

Hold Right Ctrl + Right Shift and speak. Release to transcribe. Works in any application — browser, editor, chat, email. Fully customisable hotkeys with Push-to-Talk or Toggle mode.

Groq Whisper STT

State-of-the-art speech recognition. Fast, accurate, multi-language. Supports English, French, Spanish, German and Italian.

LLM post-processing

Llama-3 fixes punctuation, removes filler words, formats bullet lists and converts spoken emoji descriptions — automatically.

Bilingual interface

Full English / French UI switchable live without restart. Dictation language is independent — transcribe in any supported language.

Context instructions

Select text, hold Right Alt + Right Shift, speak an instruction. Llama-3 rewrites or replies using your selection as context.

System tray & startup

Runs silently in the background. Minimises to tray. Optionally launches at Windows startup — one toggle in Settings.

How it works

Three keystrokes.
Your words, anywhere.

1

Place your cursor

Click in any text field — browser, Notepad, Slack, Word, VS Code. Anywhere that accepts keyboard input.

2

Hold & speak

Hold your hotkey and speak naturally. A floating overlay shows you're recording with a live VU meter.

3

Release & done

Release the key. Groq transcribes, Llama-3 cleans, and the text is pasted — all in under a second.
